STRUCTURE DES FICHIERS
-----------------------
Portfolio/
├── index.html      ← Le site complet (ouvrir dans un navigateur)
├── README.txt      ← Ce fichier
└── photos/         ← Dossier à créer (voir ci-dessous)
    ├── SAE11_AC1.jpg
    ├── SAE11_AC2.jpg
    ├── SAE12_AC1.jpg
    ...


ÉTAPE 1 — Créer le dossier photos/
------------------------------------
Crée un dossier nommé exactement "photos" (sans guillemets)
dans le même dossier que le fichier index.html.

ÉTAPE 3 — Placer les photos dans le dossier
---------------------------------------------
Copie toutes tes photos renommées dans le dossier photos/.


ÉTAPE 4 — Ouvrir le site
--------------------------
Double-clique sur index.html pour l'ouvrir dans ton navigateur.
Les photos apparaîtront automatiquement.

Si une photo est manquante, un emplacement gris s'affiche
avec le texte "Photo à ajouter" — le site reste fonctionnel.


FORMATS ACCEPTÉS
-----------------
Le site est prévu pour .jpg mais accepte aussi :
  - .jpeg
  - .png
  - .webp

Si tu utilises un format différent, change l'extension dans
le fichier index.html (chercher le nom du fichier, ex: SAE11_AC1).


NAVIGATEURS COMPATIBLES
-------------------------
Chrome, Firefox, Edge, Safari — versions récentes.
Le site est responsive (s'adapte aux téléphones et tablettes).


FONCTIONNALITÉS DU SITE
-------------------------
- Navigation fixe en haut (5 sections)
- Animations au défilement
- Onglets pour naviguer entre les 5 SAÉ
- Clic sur une photo → affichage en plein écran (lightbox)
- Touche Échap pour fermer le plein écran
- Placeholder élégant si photo absente
